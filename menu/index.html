Projeto: Menu Lateral Interativo com Overlay e Conteúdo Dinâmico

Estudo prático em HTML, CSS e JavaScript

Este projeto foi desenvolvido como caso de estudo com o objetivo de aplicar e reforçar meus conhecimentos em desenvolvimento web front-end, abordando conceitos fundamentais como manipulação de DOM, responsividade, transições visuais e interatividade com o usuário.

Tecnologias Utilizadas:

HTML5: Estrutura semântica da página, organização dos elementos do menu lateral, botões e conteúdo principal.
CSS3: Estilização completa da interface com uso de variáveis CSS (:root) para facilitar a manutenção e personalização de cores e temas. Também foram aplicadas animações e transições suaves com transition e transform para garantir uma experiência visual fluida.
JavaScript: Lógica de exibição do menu lateral com manipulação de classes (classList), controle de eventos (addEventListener) e interações com o overlay de fundo para reforçar o foco do usuário no menu ativo.

Funcionalidades e Interatividade

  • Menu Lateral Responsivo: O menu se adapta a diferentes telas, ideal para desktop e mobile.
  • Overlay de Fundo: Ao abrir o menu, um fundo semitransparente destaca o conteúdo ativo.
  • Transições Suaves: As mudanças de estado são animadas para uma experiência mais fluida.
  • Interatividade: O botão alterna entre abrir e fechar o menu lateral.

Objetivo do Projeto

    Este projeto faz parte da minha transição de carreira para a área de Tecnologia da Informação, sendo um estudo prático do curso de Desenvolvimento Web. Ele demonstra:

  • Estruturação lógica de componentes visuais.
  • Escrita de código limpo, Escrita reutilizável e organizado.
  • Compreensão de eventos e comportamento responsivo.
  • Criação de interfaces intuitivas mesmo com tecnologias base.
